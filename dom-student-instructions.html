Your DOM Assignment

👋 Hello Developer!

In this assignment, you will use JavaScript to select and change elements on a web page. This is called DOM Manipulation — and it is one of the most important skills in web development. Every website you love uses it.

You have been given a file called dom-playground.html. That file has a web page already built for you — with a profile card, buttons, lists, and more. Your job is to write JavaScript to control that page.

Before You Begin — Follow These Steps
1
Open the file in VS Code Find the file called dom-playground.html and open it in VS Code.
2
Open the same file in your browser Right-click the file in VS Code and choose "Open with Live Server" (or just drag the file into Chrome). You should see the playground page on screen.
3
Scroll to the bottom of the file In VS Code, scroll to the very bottom. You will see a <script> tag with comments like // TASK 1, // TASK 2, and so on. That is where you write all your answers.
4
Write your JavaScript, then save and refresh After writing each answer, press Ctrl + S to save, then refresh your browser to see the change happen on the page.
5
Use this sheet to write your answers too For each task below, write your answer in the dotted box provided. Your teacher will review both your code file and this sheet.
The 10 Tasks — 1 Point Each
Task 1 Change the Student Name 1 pt

On the playground page, the profile card shows the text "Student Name". Use JavaScript to change it to your own first and last name.

Which element to target
Look for the element with the id student-name on the page.
💡 Hint: Use document.getElementById('student-name') and then set its .innerHTML to your name.
Task 2 Update the Announcement 1 pt

The announcement box currently says a welcome message. Change it to say: "Assignment submitted! Well done, developer."

Which element to target
The element with the id announcement
💡 Hint: This time use document.querySelector('#announcement') and set .textContent.
Task 3 Change the Colour Box Background 1 pt

There is a grey box on the page that says "Change my background colour!". Use JavaScript to give it a background colour of your choice. Also change the text inside it to say what colour you chose.

Which element to target
The element with the id colour-box
💡 Hint: Use .style.background = 'red' to change the colour. Then use .innerHTML to change the text.
Task 4 Target the Second Subject in the List 1 pt

There is a list of subjects on the page. Select the second item only and change its text to "JavaScript DOM". Do not touch the other items.

Which element to target
The list items have the class subject-item. You want the one at index [1] (remember, counting starts at 0).
💡 Hint: Use document.querySelectorAll('.subject-item') to get all of them, then add [1] at the end to pick the second one.
Task 5 Rename the Blue Button 1 pt

The blue button currently says "Submit Work". Change its text to "Assignment Submitted ✅".

Which element to target
The button with the id btn-primary
💡 Hint: Buttons are just elements too — you can set their .innerHTML the same way.
Task 6 Reveal the Hidden Secret Box 1 pt

Somewhere on the page there is a secret box that is completely invisible. It is hidden using the CSS rule display: none. Your job is to make it appear using JavaScript.

Which element to target
The element with the id secret-box
💡 Hint: If display: none hides it, then setting .style.display = 'block' will show it.
Task 7 Update the Tasks Done Stat 1 pt

The stats section shows 0 Tasks Done. Change that number to 5 using JavaScript.

Which element to target
The element with the id stat-tasks
💡 Hint: It only contains a number, so .textContent = '5' is enough.
Task 8 Change the Banner Caption 1 pt

Below the banner image there is a caption. Update it to say: "Caption: My First DOM Assignment — Techpro 2025"

Which element to target
The element with the id image-caption
💡 Hint: You can use either getElementById or querySelector here — both will work. Try querySelector this time for practice.
Task 9 Add a New Item to the To-Do List 1 pt

The to-do list already has 5 items. Create a brand new list item using JavaScript and add it to the bottom of the list. The new item should say: "Use createElement like a pro"

Which element to target
The ul list with the id todo-list — you will add your new item inside it.
💡 Hint: This task has 3 steps — (1) create the element with document.createElement('li'), (2) set its text with .innerHTML, (3) attach it to the list with appendChild().
Task 10 Make the Green Button Show an Alert 1 pt

When a user clicks the green "Mark Done" button, an alert popup should appear saying: "Great job! Task marked as complete."

Which element to target
The button with the id btn-success
💡 Hint: Use .addEventListener('click', function() { ... }). Put your alert(...) inside the curly braces.
